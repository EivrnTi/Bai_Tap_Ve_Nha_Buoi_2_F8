Hướng Dẫn Xây Dựng Ứng Dụng Todo List Đơn Giản

Ứng dụng Todo List là một dự án nhỏ nhưng rất hữu ích để làm quen với lập trình web. Chúng ta sẽ xây dựng một ứng dụng cho phép người dùng thêm công việc, hiển thị danh sách công việc và xóa chúng khi hoàn thành.

Các bước thực hiện:

Bước 1: Tạo cấu trúc HTML (index.html)

Đầu tiên, chúng ta sẽ tạo một file index.html để định nghĩa cấu trúc cơ bản của ứng dụng. Đây là nơi chứa các phần tử giao diện như ô nhập liệu, nút thêm và danh sách các công việc. 


<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ứng dụng Todo List</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="todo-container">
        <h1>Todo List</h1>
        <div class="input-section">
            <input type="text" id="todo-input" placeholder="Thêm công việc mới...">
            <button id="add-button">Thêm</button>
        </div>
        <ul id="todo-list">
            <!-- Công việc sẽ được thêm vào đây bằng JavaScript -->
        </ul>
    </div>

    <script src="script.js"></script>
</body>
</html>

Giải thích:

  • <!DOCTYPE html> và <html>: Khai báo loại tài liệu và thẻ gốc của trang HTML.
  • <head>: Chứa các thông tin meta như charset (UTF-8 cho tiếng Việt), viewport (tối ưu hiển thị trên các thiết bị), tiêu đề trang (<title>) và liên kết đến file CSS (<link rel="stylesheet" href="style.css">).
  • <body>: Chứa nội dung hiển thị của trang.
  • <div class="todo-container">: Một thẻ div bao bọc toàn bộ ứng dụng, giúp dễ dàng tạo kiểu dáng bằng CSS.
  • <h1>Todo List</h1>: Tiêu đề chính của ứng dụng.
  • <div class="input-section">: Chứa ô nhập liệu (<input type="text" id="todo-input">) và nút thêm (<button id="add-button">). Các id này sẽ được sử dụng bởi JavaScript để thao tác.
  • <ul id="todo-list">: Một danh sách không thứ tự nơi các công việc sẽ được hiển thị.
  • <script src="script.js"></script>: Liên kết đến file JavaScript, được đặt ở cuối <body> để đảm bảo các phần tử HTML đã được tải trước khi script thực thi.

Bước 2: Tạo kiểu dáng với CSS (style.css)

Tiếp theo, chúng ta sẽ tạo một file style.css để làm cho ứng dụng trông đẹp mắt hơn. Điều này bao gồm việc định dạng các phần tử, màu sắc và bố cục. 


body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    display: flex;
    justify-content: center;
    align-items: center;
    min-height: 100vh;
    margin: 0;
}

.todo-container {
    background-color: #fff;
    padding: 30px;
    border-radius: 8px;
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);

    text-align: center;
}

h1 {
    color: #333;
    margin-bottom: 20px;
}

.input-section {
    display: flex;
    margin-bottom: 20px;
}

#todo-input {
    flex-grow: 1;
    padding: 10px;
    border: 1px solid #ddd;
    border-radius: 4px 0 0 4px;
    outline: none;
}

#add-button {
    padding: 10px 15px;
    background-color: #007bff;
    color: white;
    border: none;
    border-radius: 0 4px 4px 0;
    cursor: pointer;

}

#todo-list li button:hover {
    background-color: #c82333;
}

Giải thích:

  • body: Định nghĩa font chữ, màu nền, và sử dụng Flexbox để căn giữa nội dung ứng dụng trên trang. min-height: 100vh; đảm bảo body chiếm toàn bộ chiều cao khung nhìn.
  • .todo-container: Thẻ bao bọc chính, được tạo kiểu dáng với màu nền trắng, padding, bo tròn góc, và đổ bóng để tạo hiệu ứng nổi bật.
  • h1: Đặt màu chữ cho tiêu đề.
  • .input-section: Sử dụng Flexbox để sắp xếp ô nhập liệu và nút thêm trên cùng một hàng.
  • #todo-input: Ô nhập liệu được làm cho đầy đủ chiều rộng còn lại (flex-grow: 1;), có padding và viền.
  • #add-button: Nút "Thêm" được tạo màu xanh lam, có hiệu ứng chuyển màu khi di chuột qua.
  • #todo-list: Xóa bỏ dấu đầu dòng mặc định của danh sách (list-style: none;) và padding.
  • #todo-list li: Mỗi mục công việc (li) có màu nền, padding, bo tròn góc, và sử dụng Flexbox để căn giữa nội dung và nút xóa, đồng thời word-break: break-word; giúp văn bản dài xuống dòng.
  • #todo-list li button: Nút xóa bên trong mỗi công việc được tạo màu đỏ, có hiệu ứng hover.

Bước 3: Thêm chức năng với JavaScript (script.js)

Cuối cùng, chúng ta sẽ tạo file script.js để thêm các chức năng tương tác cho ứng dụng: thêm công việc, hiển thị và xóa công việc. 


document.addEventListener('DOMContentLoaded', function() {
    const todoInput = document.getElementById('todo-input');
    const addButton = document.getElementById('add-button');
    const todoList = document.getElementById('todo-list');

    // Hàm thêm công việc
    function addTodo() {
        const todoText = todoInput.value.trim(); // Lấy giá trị và loại bỏ khoảng trắng thừa
        if (todoText === '') {
            alert('Vui lòng nhập công việc!');
            return;
        }

        // Tạo thẻ li mới
        const listItem = document.createElement('li');

        // Tạo span chứa nội dung công việc
        const todoSpan = document.createElement('span');
        todoSpan.textContent = todoText;

        // Tạo nút xóa
        const deleteButton = document.createElement('button');
        deleteButton.textContent = 'Xóa';
        deleteButton.addEventListener('click', function() {
            todoList.removeChild(listItem); // Xóa mục công việc khỏi danh sách
        });

        // Thêm span và button vào li
        listItem.appendChild(todoSpan);
        listItem.appendChild(deleteButton);

        // Thêm li vào danh sách ul
        todoList.appendChild(listItem);

        // Xóa nội dung trong ô nhập liệu
        todoInput.value = '';
    }

    // Gắn sự kiện click cho nút Thêm
    addButton.addEventListener('click', addTodo);

    // Gắn sự kiện Enter cho ô nhập liệu
    todoInput.addEventListener('keypress', function(event) {
        if (event.key === 'Enter') {
            addTodo();
        }
    });
});

Giải thích:

  • document.addEventListener('DOMContentLoaded', function() { ... });: Đảm bảo mã JavaScript chỉ chạy sau khi toàn bộ tài liệu HTML đã được tải và phân tích cú pháp.
  • const todoInput = document.getElementById('todo-input');: Lấy tham chiếu đến phần tử ô nhập liệu bằng id của nó.
  • const addButton = document.getElementById('add-button');: Lấy tham chiếu đến nút "Thêm".
  • const todoList = document.getElementById('todo-list');: Lấy tham chiếu đến danh sách <ul>.
  • Hàm addTodo():
    • todoInput.value.trim(): Lấy giá trị từ ô nhập liệu và dùng .trim() để loại bỏ các khoảng trắng ở đầu và cuối chuỗi.
    • Kiểm tra nếu todoText rỗng, hiển thị cảnh báo và thoát hàm.
    • document.createElement('li'): Tạo một thẻ <li> mới.
    • document.createElement('span'): Tạo một thẻ <span> để chứa nội dung công việc.
    • todoSpan.textContent = todoText;: Gán nội dung công việc cho <span>.
    • document.createElement('button'): Tạo một thẻ <button> cho chức năng xóa.
    • deleteButton.addEventListener('click', function() { ... });: Gắn sự kiện click cho nút xóa. Khi click, nó sẽ gọi todoList.removeChild(listItem); để xóa mục <li> tương ứng khỏi danh sách.
    • listItem.appendChild(...): Thêm <span> và nút xóa vào <li>.
    • todoList.appendChild(listItem);: Thêm <li> mới tạo vào danh sách <ul>.
    • todoInput.value = '';: Xóa nội dung trong ô nhập liệu sau khi thêm công việc.
  • addButton.addEventListener('click', addTodo);: Gắn hàm addTodo vào sự kiện click của nút "Thêm".
  • todoInput.addEventListener('keypress', function(event) { ... });: Gắn sự kiện keypress (nhấn phím) cho ô nhập liệu. Nếu phím được nhấn là Enter (event.key === 'Enter'), hàm addTodo sẽ được gọi, giúp người dùng thêm công việc nhanh chóng bằng cách nhấn Enter.

Hoàn thành!

Vậy là bạn đã hoàn thành một ứng dụng Todo List đơn giản với đầy đủ chức năng thêm, hiển thị và xóa. Bạn có thể mở file index.html trong trình duyệt để xem kết quả. Hãy tiếp tục khám phá và tùy chỉnh để làm cho ứng dụng trở nên phong phú hơn nhé!
